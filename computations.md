* prędkość blatu / obroty
Dv = 5.5 mm/rot

* pożądana prędkość blatu

Vb = 25 - 36 mm/s

* zależność prędkości blatu / obroty z obrotami silnika (Ev)

Vb [mm/s] = Dv [mm/rot] * Ev [rot/s]

* wyliczenie Ev
Ev = Vb / Dv


1. 25 mm/s
Ev1 = 25 mm/s / 5.5 mm/rot ~= 5Hz = 300rpm
Ev2 = 35 mm/s / 5.5 mm/rot ~= 7Hz = 420rpm


powiedzmy że trzeba przemieścić 500mm
Dv = L / R
L = 500mm
R = liczba obrotów = L / Dv = 500 / 5.5 ~= 100
R = Ev * t
Dv = L / Ev / t
Ev = L / Dv / t
Ev = 500 / 5.5 / 60 (*60) = 90RPM
